# Sprint Review

Run Phase 3 of Sprint Protocol.

1. Read `../sprint-protocol/SKILL.md`.
2. Read `../sprint-protocol/references/codex-orchestration.md`.
3. Read `../sprint-protocol/references/phase-3-review.md`.
4. Use `update_plan`.
5. Review user feedback, story sizing, dependencies, and compliance.
6. Spawn auditors or story revision workers as needed.
7. Audit required skills, feature-level tests, difficulty/model routing, branch assumptions, and commit strategy.
8. Produce `verification-checklist.md`.

Treat legacy `/sprint-review <name>` as this skill.
